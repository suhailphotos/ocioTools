This package defines the default OpenColorIO configuration.

These files are built from the AcademySoftwareFoundation
OpenColorIO-Config-ACES project:
    https://github.com/AcademySoftwareFoundation/OpenColorIO-Config-ACES

Additional OCIO configuration files can be found at:
    http://opencolorio.org/configurations/

Further information about OpenColorIO may be found at:
    http://opencolorio.org/
